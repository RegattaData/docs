---
title: Object Relational Mapping Tools (ORM)
---

Regatta supports the following ORM tools:

- TypeORM
- Hibernate
To download an ORM tool or get the full list of available ORM tools, please reach out to Regatta 
support.

<Note> Note: Whenever trying to execute an operation which is not supported by the Regatta dialect, 
the driver will throw an Unsupported Feature exception in response. </Note>
