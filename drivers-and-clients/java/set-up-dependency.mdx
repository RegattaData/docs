---
title: Setting Up Dependency
---

Now you need to add the Regatta JDBC dependency to your pom.xml:

```java
<dependency>
    <groupId>dev.regatta</groupId>
    <artifactId>jdbc</artifactId> 
    <version>1.0</version>
    <scope>system</scope>
    <systemPath>my_drivers/regatta_jdbc.jar</systemPath>
</dependency>
```
