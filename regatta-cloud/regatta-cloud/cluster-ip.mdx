---
title: Cluster IP
---

To attain a cluster IP, go to the platform and access the desired database by clicking on its name 
in the **Cluster Name** column in the **Clusters** page. You will notice that it is a link. 

The following page will be displayed:

<Frame caption="Figure 44 Regatta Cluster IP">
  <img src="/images/43.png" />
</Frame>

Under the **Security** section in the **Connection Details** tab, you will see the IP and PORT of that 
cluster. Copy them for the next steps.
