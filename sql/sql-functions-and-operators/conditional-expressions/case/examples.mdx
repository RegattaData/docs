#### `CASE` command using the first syntax above:

```sql
SELECT *,
CASE
WHEN o_totalprice >= 1000000 THEN o_totalprice*0.20
WHEN o_totalprice >= 500000 THEN o_totalprice*0.10
WHEN o_totalprice >= 100 THEN o_totalprice
END AS price_after_discount
FROM orders;
```

#### `CASE` command using the second syntax above:

```sql
SELECT c_name,
CASE c_gender
WHEN 'F' THEN 'Female'
WHEN 'M' THEN 'Male'
ELSE NULL
END AS customer_gender
FROM customers;
```

#### Using `CASE` in the `WHERE` clause:

```sql
SELECT COUNT(*),
SUM(o_totalprice),
o_name
FROM orders, customers
WHERE o_custkey=c_custkey
AND CASE WHEN o_name='this is first expression'
THEN o_date>'2020-01-01'
WHEN o_name='this is the second expression'
THEN o_date<'2020-01-01'
ELSE o_date IS NULL
END
GROUP BY o_name;
```
