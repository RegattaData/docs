---
title: "Manual Deployment"
---

The following sections describes how to perform a manual deployment of a Regatta Cluster.

It will consist of the following sections:

1. [User Setup](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/user-setup) - Setting up the deployment user
2. [Regatta Configuration](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/regatta-configuration/regatta-configuration) - Selecting the different modules that construct the Regatta database
3. [Configurating Regatta](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/starting-the-system) - How to actually configure the Regatta database
4. [Starting the System](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/starting-the-system) - Run the database instance
5. [Cli and Drivers ](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/cli-and-drivers)- Connecting and working with the Regatta database
