---
title: "Manual Deployment"
---

The following sections describe how to perform a manual deployment of a Regatta Cluster.

It will consist of the following sections:

1. [User Setup](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/user-setup) - Setting up a dedicated user for deploying and running Regatta
2. [Regatta Software Modules](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/regatta-software-modules) - Selecting the different modules that construct the Regatta database
3. [Firewall](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/firewall) - Configuring the nodes firewall to allow Regatta modules inter-node communication
3. [Configurating Regatta](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/configuration-file) - How to create a configuration file for the Regatta database
4. [Starting the System](https://docs.regatta.dev/self-hosted-deployment/manual-deployment/starting-the-system) - Run the database instance
