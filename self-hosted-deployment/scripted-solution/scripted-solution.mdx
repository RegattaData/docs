---
title: "Scripted Solution"
---

Regatta offers a scripted solution that will set up the Regatta cluster based on minimal user input. The script follows the same method described in the manual process with the recommended configuration.

The script can be downloaded from a link that will be supplied from Regatta Support. The download will include a tar archive containing:

1. The RPM packages
2. A folder holding the scripted solution with all its components

Once downloaded, continue with the steps in the following pages.
