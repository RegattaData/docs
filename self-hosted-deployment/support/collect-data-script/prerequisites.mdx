---
title: "Prerequisites"
---

To use the script, you will need the following:

1. **From Regatta Support**

   1.1 The **collect_data** script

   
   1.2 Storage bucket details for data upload

   
   1.3 A `service_account_json_file` with authorization to access the bucket

2. **Dependencies**

   2.1 **python3** version 3.6.8 or later

   
   2.2 **google-cloud-storage** Python module, version 2.0.0 or later

   
   2.3 **zstandard** Python module, version 0.20.0 or later

3. **Network Requirements**

   3.1 Outbound internet connection to upload the collected data

<Note>
  Some prerequisites may be optional depending on the method chosen for collecting the data.
</Note>
