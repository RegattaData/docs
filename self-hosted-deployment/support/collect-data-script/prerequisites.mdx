---
title: "Prerequisites"
---

To use the script, you will require the following:

1. From Regatta support:

   a. The **collect_data** script

   b. The storage bucket particulars for data upload

   c. A service_account_json_file with authorization to access the bucket
2. Dependencies:

   a. **python3 **version 3.6.8 and above 

   b. **google-cloud-storage** python module, version 2.0.0 and above

   c. **zstandard** python module, version 0.20.0 and above 
3. Outbound Internet connection to upload the collected data

<Note>
   Note that some prerequisites will be optional depending on the manner you decide to collect
  the data. 
</Note>
